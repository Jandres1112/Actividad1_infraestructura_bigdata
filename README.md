# Actividad 1 - Infraestructura Big Data

Proyecto académico orientado a la implementación de un flujo básico de extracción, almacenamiento y procesamiento de datos utilizando herramientas de ingeniería de datos.

---

## Objetivo

Construir una pequeña arquitectura de datos que permita:

- Extraer información desde una fuente externa (API o scraping).
- Almacenar datos en base de datos local.
- Generar archivos procesados (.csv, logs).
- Documentar dependencias y entorno de ejecución.
